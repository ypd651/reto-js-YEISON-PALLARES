# 🚀 Reto JS - YEISON PALLARES

Proyecto desarrollado en JavaScript, HTML y CSS como parte de un reto práctico de programación frontend.

## 📋 Descripción

Este proyecto consiste en una aplicación web construida con tecnologías base del frontend:

- HTML5
- CSS3
- JavaScript Vanilla

El objetivo principal es practicar la manipulación del DOM, manejo de datos JSON y lógica de programación en JavaScript.
